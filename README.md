# FreeTools

**Access System** <br/>
``0x1`` : Porte Ouverte <br/>
``0x2`` : Porte Fermée <br/>
``0x3`` : Mouvement Détecté <br/>
``0x4`` : Intru dans l'abri <br/>
``0x5`` : Utilisateur dans l'abri <br/>
 <br/>
**Tools_System** <br/>
``1x1`` : Outil disponible <br/>
``1x2 <dd/mm/yy>`` : Date de Retour <br/>
``1x3 <dd>`` : Combien de jour restant
